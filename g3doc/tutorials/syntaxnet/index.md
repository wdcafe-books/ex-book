# SyntaxNet
(v0.9)

## 소개

SyntaxNet는 텐서플로우를 위한 뉴럴 네트워크 방식의 자연어 처리 프레임워크입니다.

## SyntaxNet의 기본 튜토리얼

해당 [튜토리얼](
https://github.com/tensorflow/models/tree/master/syntaxnet#installation)
은 다음 항목들을 설명하고 있습니다:

   *   SyntaxNet를 설치하는 방법
   *   SyntaxNet 패키지에 포함된 Parsey McParseface 파서를 사용하는 방법
   *   사용자의 품사 태거를 훈련하는 방법
   *   사용자의 파서를 훈련하는 방법
